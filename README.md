# 🌐 Shweta Rani | Developer Portfolio

A modern, responsive, and interactive developer portfolio showcasing my projects, technical skills, achievements, certifications, and coding journey. Built to reflect my passion for full-stack development, problem-solving, and creating scalable web applications.

🔗 **Live Link:** [portfolio-shweta-lyart.vercel.app](https://portfolio-shweta-lyart.vercel.app/)

## Features

*  Modern and responsive UI with smooth animations
*  **Page Likes System:** Interactive heart button with a live like count and localStorage-based persistence
*  **Contact Form:** Secure and verified form submissions routed directly to email using Resend
*  Professional introduction and career overview
*  Featured full-stack and frontend projects
*  Technical skills and technology stack
*  Achievements, certifications, and coding milestones
*  LeetCode statistics and coding journey
*  Dark/Light theme support
*  Fast performance with optimized Next.js architecture

##  Tech Stack

* **Framework:** Next.js 16 (Turbopack)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **UI Components:** shadcn/ui
* **Email Service:** Resend
* **Icons:** Lucide React
* **Deployment:** Vercel
